### NewsAggregator
NewsAggregator allows you to view top posts from Reddit and HackerNews.

### Tech
* ASP.NET Core 5.0
* EF Core 5.0
* Identity Auth with JWT
* Hangfire

### Requirements
* Dotnet SDK

### Usage
* WebAPI
  * In the project root directory run
  ```
  dotnet build
  ```
  * then
  ```
  dotnet run
  ```
  App starts on http://localhost:5000
